# Read a Data Package descriptor file (`datapackage.json`)

Read a Data Package descriptor file (`datapackage.json`)

## Usage

``` r
read_package(file = "datapackage.json")
```

## Arguments

- file:

  Path or URL to a `datapackage.json` file.
