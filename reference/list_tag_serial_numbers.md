# List all available tag serial numbers

List all available tag serial numbers

## Usage

``` r
list_tag_serial_numbers(connection)
```

## Arguments

- connection:

  **\[deprecated\]** A connection to the ETN database. This argument is
  no longer used. You will be prompted for credentials instead.

## Value

A vector of all unique `tag_serial_numbers` present in
`common.tag_device`.
