# List all available animal project codes

List all available animal project codes

## Usage

``` r
list_animal_project_codes(connection)
```

## Arguments

- connection:

  **\[deprecated\]** A connection to the ETN database. This argument is
  no longer used. You will be prompted for credentials instead.

## Value

A vector of all unique `project_code` of `type = "animal"` in
`project.sql`.
