# List all available acoustic tag ids

List all available acoustic tag ids

## Usage

``` r
list_acoustic_tag_ids(connection)
```

## Arguments

- connection:

  **\[deprecated\]** A connection to the ETN database. This argument is
  no longer used. You will be prompted for credentials instead.

## Value

A vector of all unique `acoustic_tag_id` in `acoustic_tag_id.sql`.
